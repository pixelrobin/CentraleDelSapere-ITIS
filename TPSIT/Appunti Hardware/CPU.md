# La CPU

La CPU è considerata il cervello del computer e puó avere diverse forme

Pin Grid Array (PGA): in questa architettura i pin sono sotto la CPU e viene inserito nel socket della CPU senza forza di spinta (ZIF zero insertion force)

![[XC68020_bottom_p1160085 1.jpg|350]]

Land Grid Array (LGA): in questa architettura i pin sono nel socket

![[Intel_CPU_Pentium_4_640_Prescott_bottom.jpg|350]]

La CPU é connessa al [Northbridge](obsidian://open?vault=Vault%20di%20denis&file=Informatica%2FAppunti%20Hardware%2FNorthbridge%20e%20Soutbridge)
