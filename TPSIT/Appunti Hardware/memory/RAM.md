# Introduzione

La RAM o Random Access Memory é la memoria centrale di un computer. E' una memoria volatile per i programmi in uso dalla CPU. Volatile  significa che ogni volta che si spegne il computer i dati vengono cancellati. Con meno RAM, il computer deve swappare ogni volta tra la RAM e gli hard drive o SSD.

--- 
## Tipi di RAM

**DRAM** (Dynamic RAM) --> Usata nel passato come memoria centrale, deve essere costantemente aggiornata con un impulso elettrico

**SRAM** (Static RAM) --> Richiede energia costante, usata per la memoria [[Cache]], molto piú veloce delle DRAM ma piú costose.

![[1692861675331-1-01 (69).png]]

**SDRAM** (Syncronous DRAM) --> DRAM che operano in sincronizzazione con il memory bus, maggior velocità di trasmissione.

![[RAM_module_SDRAM_1GiB.jpg]]

**DDR RAM** (Double Data Rate RAM) --> trasferisce i dati il doppio piú veloce delle SDRAM

**DDR2** --> maggior  velocitá del clock

**DDR3** --> usa meno energia delle DDR2 e genera meno calore, maggior velocitá di clock

**DDR4** --> quadruplica la capacitá di memoria delle DDR3, consuma meno potenza, molta piú velocitá di clock

**GDDR** (Graphics Double Data Rate) --> sono le RAM per le GPU

**DDR5** --> piú del doppio della velocitá di clock delle DDR4, 4x volte la capacitá di memoria delle DDR4. Un singolo stick può arrivare a 128GB.
