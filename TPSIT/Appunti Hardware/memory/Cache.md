# Introduzione

La [SRAM](obsidian://open?vault=Vault%20di%20denis&file=TPSIT%2FAppunti%20Hardware%2Fmemory%2FRAM) è di solito la memoria piú veloce ed è usata per la cache che salva le istruzioni e i dati piú recenti della [CPU](obsidian://open?vault=Vault%20di%20denis&file=TPSIT%2FAppunti%20Hardware%2FCPU) ed è divisa in 3 livelli:

---

- **Cache L1**: Integrata nella CPU, é la cache piú veloce ma anche quella con meno spazio.
- **Cache L2**: cache esterna ed è era sulla scheda madre vicino alla CPU, ora è integrata nella CPU. Ha una memoria leggermente maggiore della L1.
- **Cache L3**: usata nelle workstation di fascia alta sulle CPU dei server. É la cache con piú memoria ma é anche la piú lenta.
