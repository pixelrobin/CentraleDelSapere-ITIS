# Introduzione

Gli errori di memoria si verificano quando i dati non vengono salvati nel modo corretto nei chip.

---

Esistono 3 tipi di memorie per gli errori:

==Nonparity==: la memoria di non paritá **non verifica gli errori** presenti in memoria, la RAM non di paritá è la RAM piú comune sul mercato.

==Parity==: contiene 8 bit per i dati e un bit per controllare gli errori, il bit che controlla gli errori si chiama **parity-bit**

==ECC==: Error Correction Code puó individuare **errori multipli** in memoria ma puó **correggere** **errori singoli** in memoria, usato nei server finanziali e di grandi aziende di data anlysis.
