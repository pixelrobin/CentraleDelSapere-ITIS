### 🎯 Obiettivo del Progetto

Questo repository è nato per **superare la frammentazione delle conoscenze** e creare una base di sapere condivisa, accessibile e in continua crescita.

L'obiettivo è costruire un **"secondo cervello" collettivo** dove ogni studente può contribuire, consultare e migliorare gli appunti delle materie scolastiche, applicando metodologie di sviluppo software (come il versionamento con Git) alla gestione del nostro apprendimento.

---

### 🛠️ Stack Tecnologico

| Strumento | Scopo |
| :--- | :--- |
| **Obsidian** | ✍️ Per scrivere e organizzare gli appunti in formato Markdown, creando una rete di collegamenti tra i concetti. |
| **Git & GitHub** | 🔄 Per il controllo di versione e la collaborazione asincrona, permettendoci di lavorare insieme senza conflitti. |
| **Syncthing** | ☁️ Utilizzato a livello individuale per la sincronizzazione privata e sicura del vault su più dispositivi. |

### 🚀 Guida Rapida all'Installazione

Segui questi passaggi per avere la "Centrale del Sapere" pronta sul tuo PC in pochi minuti.

1.  **Clona il Repository**
    Scarica il vault sul tuo computer con il seguente comando Git:
    ```bash
    git clone https://github.com/pixelrobin/CentraleDelSapere-ITIS.git
    ```

2.  **Apri in Obsidian**
    - Apri Obsidian.
    - Seleziona `Apri un'altra cartella come vault`.
    - Scegli la cartella `CentraleDelSapere-ITIS` che hai appena clonato.

3.  **Sincronizza le Modifiche**
    Per mantenere il tuo vault locale sempre aggiornato, esegui questo comando **prima di iniziare a lavorare**:
    ```bash
    git pull
    ```

### 🤝 Come Contribuire

Contribuire è semplice, ma per mantenere l'ordine è fondamentale seguire delle linee guida.

1.  **Crea o Modifica Note**
    - Aggiungi nuovi appunti o migliora quelli esistenti.
    - **Importante**: Prima di iniziare, assicurati di aver letto le nostre **[linee guida per la formattazione](./LINEE_GUIDA.md)**.

2.  **Salva le Tue Modifiche**
    Una volta terminato, salva il tuo lavoro sul repository condiviso con questi comandi:
    ```bash
    # Aggiunge tutte le modifiche fatte
    git add .

    # Crea un "commit" con un messaggio che descrive cosa hai fatto
    git commit -m "Tuo messaggio (es: Aggiunti appunti di Sistemi su TCP/IP)"

    # Carica le modifiche su GitHub
    git push
    ```
